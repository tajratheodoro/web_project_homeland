# Projeto TripleTen Sprint 5: TripleTen Art Gallery - De Pátria para Pátria

# Focado em desenvolver habilidades adaptação do layout para diferentes tamanhos de tela (responsividade) com uam homeland de uma página web.

Tecnologias utilizadas:
**Layout Responsivo com Media Queries: A página se adapta perfeitamente a desktops, tablets e dispositivos móveis**
**Flexbox para alinhamento eficiente dos elementos da página**
**Metodologia BEM para nomeclatura**
**Importação de fontes localmente da fonte "Inter" para tipografia moderna.**
